What is Hackedbox?

Hackedbox is a stripped down version of Blackbox - The X11 Window Manager (http://blackboxwm.sourceforge.net/). The toolbar and Slit have been removed. The goal of Hackedbox is to be a small "feature-set" window manager, with no bloat. There are no plans to add any functionality, only bugfixes and speed enhancements whenever possible.

Hackedbox is only available in source form. Package submissions are welcome. Please submit them to <larry@scrudgeware.org>.

Downloading

Available at the Scrudgeware ftp site:ftp://scrudgeware.org/hackedbox-0.7.0.tar.gz

Compiling

Nothing tricky:

  • ./configure

  • make

  • make install

Then edit ~/.xinitrc to make "hackbox" your window manager. Or whatever way you use to switch, gnome users can use the control panel.
